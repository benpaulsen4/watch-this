Import lets you restore data from a previous WatchThis export. It’s designed for migrations and backups.

## Before You Start

- Imports currently support **JSON exports only**.
- Import is additive, which means it adds or updates records rather than wiping your account.

If you import multiple times, you may end up with duplicate list items.

## Import a JSON Export

1. Go to **Profile** (`/profile`).
2. Open the **Data** tab.
3. Under **Import Data**, click **Choose File**.
4. Select your `.json` export.
5. Click **Import Data**.

After the import finishes, WatchThis shows how many lists, items, statuses, episodes, and schedules were processed.

## Tips

- For the cleanest migration, import into a new (empty) account.
- If you only want to restore some lists, consider editing the JSON file before importing.
