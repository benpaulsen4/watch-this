You can add or remove movies and TV shows from lists without leaving what you’re currently doing. The fastest way is through the title’s details view.

## Add Content to an Existing List

1. Open a movie or TV show’s details (from Discover or a list).
2. Go to the **Lists** tab.
3. Under **Manage Lists**, find the list you want.
4. Click **Add**.

If the title is already in a list, you’ll see **Added**.

## Remove Content from a List

1. Open the title’s details.
2. Go to the **Lists** tab.
3. Find the list where the title is currently included.
4. Click **Remove**.

Tip: When you open details from inside a list, WatchThis marks that list as **Current**, which makes it easy to remove the item you’re looking at.

## Add to a New List (Shortcut)

If you don’t have the right list yet:

1. Open the title’s details → **Lists** tab.
2. Click **Add to new list**.
3. Create the list.

WatchThis creates the list and adds the title immediately.

## Common Issues

### “No compatible lists found”

This means all your existing lists are the wrong type for that title (for example, trying to add a TV show to a Movies Only list). Create a compatible list type and try again.

Read: [Create and Manage Lists](/help/lists/Create%20and%20Manage%20Lists).
