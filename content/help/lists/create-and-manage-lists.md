Lists are how you organize movies and TV shows in WatchThis. You can keep lists private, share them with collaborators, or make them public so others can view them.

## Create a List

1. Go to **My Lists** (`/lists`).
2. Click **Create List**.
3. Fill out the list settings:
   - **List Name** (required)
   - **Description** (optional)
   - **List Type** (Mixed, Movies Only, TV Shows Only)
   - **Make this list public** (optional)
4. Click **Create**.

## Understand List Types

List types help keep lists focused:

- **Mixed (Movies & TV Shows)**: can contain both.
- **Movies Only**: only movie titles can be added.
- **TV Shows Only**: only TV shows can be added.

If you try to add incompatible content, WatchThis will ask you to choose a compatible list (or create a new one).

## Edit List Settings

1. Open the list.
2. Click **Settings**.
3. Update the fields you want.
4. Click **Save Changes**.

Only the list owner can change list settings.

## Share a List

To invite collaborators, open the list and click **Share**.

Read: [Share a List with Collaborators](/help/collaboration/share-a-list-with-collaborators).

## Sync Watch Status (Shared Lists Only)

If you share a list, you can optionally enable **Sync watch status & TV schedules with collaborators** in list settings.

This syncs progress for titles that are in that shared list.

Read: [How Sync Works in Shared Lists](/help/collaboration/how-sync-works-in-shared-lists).
